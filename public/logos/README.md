# Logos para Atribución

Esta carpeta contiene los logos de los proveedores de datos usados en GeekHub.

## Logos requeridos

### TMDb (The Movie Database)

- **Archivo:** `tmdb.svg`
- **Descarga:** https://www.themoviedb.org/about/logos-attribution
- **Uso:** Atribución obligatoria para datos de películas y series

### RAWG (Video Games Database)

- **Archivo:** `rawg.svg`
- **Descarga:** https://rawg.io/apidocs (ver sección branding)
- **Uso:** Atribución obligatoria para datos de videojuegos

## Instrucciones

1. Descarga los logos oficiales desde los enlaces anteriores
2. Guárdalos en esta carpeta con los nombres exactos (`tmdb.svg`, `rawg.svg`)
3. Los logos deben ser SVG preferiblemente, o PNG con fondo transparente

## Notas

- Si los logos no están presentes, la aplicación mostrará un fallback de texto
- No se incluyen los logos en el repositorio por restricciones de licencia
- Asegúrate de cumplir con los términos de uso de cada proveedor
